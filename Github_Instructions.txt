Instructions for GitHub on Windows:
1) Setup an account
2) Send Juan_Cortez@utexas.edu your email so he can add you as a collaborator
3) Follow instructions on this page: https://help.github.com/articles/set-up-git/
4) Clone the repository: https://github.com/juancortez-ut/mppt.git

These are the commands you will be using on github:

CMD						EXPLANATION
====================================================================
git pull					Gets latest code
git status					What files have changed
git add -A					Adds everything to Master
git commit <file destination> -m "comment"	Commit files
git push					Commit Everything to Master
